<h3>传统数仓和大数据数仓的区别- NO.4</h3>

```python
1.存储方式：传统数仓是基于 oracle/mysql 等传统数据库的，大数据数仓是居于 Hadoop 平台中的 hdfs 等实现的
2.ETL工具：传统数仓是基于 kettle 进行处理的，但是随着数据量得到增大，kettle 变得不稳定。大数据数仓是基于 hadoop 
  中的 hive 作为数据处理工具。
```
